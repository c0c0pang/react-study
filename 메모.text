jsx는 js + xml 임.
js와 jsx의 차이?
jsx는 선형형: 결과물을 선언
js는 명령형: 어떻게 만들지 방법을 선언  
MPA = 여러개의 페이지로 구성된 어플리케이션
단점 = 상태 유지의 어려움, 불필요한 로딩
SPA = 하나의 페이지로 구성된 어플리케이션 
라우팅 = 다른 주소의 다른 화면을 보여줌 

navigate를 이용하여 다른 주소로 넘어가게 해줌.
useCallback 함수는 연산이 많은 경우 최적화를 시켜줌.
React.memo 는 컴포넌트를 memorize 하여 필요한 상황에 re-render --> 정적인 컴포넌트에 주로 사용
